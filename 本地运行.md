后端本地运行步骤：

1. 安装依赖


cd ai-interview-backend
pip install -r requirements.txt
2. 配置环境变量
将 .env.example 复制为 .env，修改以下关键配置：

DEEPSEEK_API_KEY：填入你的 DeepSeek API Key（必填）
POSTGRES_HOST=localhost，POSTGRES_PORT=5434
REDIS_HOST=localhost，REDIS_PORT=6382
3. 启动数据库和 Redis（Docker）


docker compose -f docker-compose.yml -f docker-compose.dev.yml up postgres redis -d
4. 执行数据库迁移
alembic upgrade head
5. 创建管理员账号
python scripts/create_first_admin.py
6. 启动后端
python main.py
访问 http://localhost:8006/docs 查看接口文档。
